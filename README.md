# Star Wars Quiz

A fun, interactive Star Wars trivia game built with TypeScript and Vite.

## 🎮 Features

- 10 Star Wars trivia questions.
- Multiple-choice answers.
- Score tracking and progress display.
- Results page with personalized tier ranking (Youngling, Padawan, Jedi, etc.)
- Yoda-themed UI with Star Wars styling.

 ## 🛠️ Tech Stack

- TypeScript
- Vite
- SCSS
- HTML5
